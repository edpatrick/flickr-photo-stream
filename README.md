# flickr-photo-stream

This is a simple single page app to search the photos using the Flickr API.

Features include:

* lazy loading
* infinite scroll
* search input

## Install

```
git clone https://github.com/edpatrick/flickr-photo-stream.git

cd flickr-photo-stream/

npm i
```

## Run on dev

```
npm start
```

# Build for prod

```
npm run build
```
